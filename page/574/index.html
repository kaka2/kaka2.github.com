Gtalk與FriendFeed的互動

11/11/08

FriendFeed是個人信息聚合平臺，算是SNS范疇的應用吧。每次查看FriendFeed頁面信息的更新，總是會奔首頁而去，覺得等待加載頁面的時間有些麻煩。不過現在可方便了，可以通過Gtalk客戶端來實現與FriendFeed的互動，比如：發送個人信息，接受和評論等。我們所做的就是出除了熟悉幾個操作命令外，還要在個人設置頁面激活相關選項。

官方消息：Instant FriendFeed
（Notifications and Posting over IM）

在“IM Settings”選項中激活Gmail帳戶：

在“FriendFeed Over IM”頁面找到命令句法：

在Gtalk中顯示FriendFeed的聊天機器人：ff@chat.friendfeed.com，其他命令：

1）@me text - post to your feed
2）@me - list entries in your feed
3）@room text - post to room
4）@room - list entries in room
5）@N text - comment on entry N
6）@N - show entry N
7）@user - list entries in user's feed
8）like N - like entry N
9）pause - Pause notifications
10）resume - Resume notifications
11）list - List recent entries
12）help - this command

其實不記得那麼多語法也沒關系啊，只需要記得輸入第12）個“help”就可以獲取即時幫助信息。

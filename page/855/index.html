筆記-微軟360度

12/19/07

在網絡里流傳了很多關于谷歌或百度等公司的企業和文化，但是微軟方面的卻見的很少。不像谷歌的粉絲們經常有一些專門的blog報導和資訊。

最近看《微軟360度》——企業和文化，由一批微軟華人員工撰寫的，因為這本書是按微軟公司對軟件開發和項目管理的步驟來進行一本書的“開發”的，從LiveMeeting,SharePoint等內部工具的應用=>Alpha試用版=>出版社評審=>RC版=>Gold Release，可謂開創了寫書的新篇章。

書名大概來自微軟的XBOX 360,所以叫微軟360，讓我在這里只做些簡單的筆記吧：

1）是什么吸引全世界那么多精英來到微軟公司工作？在第五章中介紹了《育才的沃土》——關于她的員工職務、工作環境、創新和創造、團隊精神等等。史蒂夫.鮑爾默自豪地說：”微軟就像一架巨大的強力真空吸收器，幾乎將地球上的信息技術(IT)精英吸納一空。“

2）微軟的宗旨：激發個人潛能，實現企業潛力。微軟第一個使命是：”讓每一張桌子上和每一個家庭里都有一臺個人電腦“。第二個使命是：“軟件無處不在。”現在的使命是：“幫助全球的個人用戶和企業展現他們所有的潛力。”

3）微軟員工在全球共使用30多萬臺個人電腦數萬個Pocket Pc、SmartPhone以及上千臺蘋果電腦（Macintosh）（微軟有一個蘋果部門專門生產蘋果辦公室套裝軟件）。為員工提供博客網站服務如：MSDN微軟開發人員網絡；TechNet:IT專業人士網。

4）新員工進入公司之前，員工的個人數據，比如工號工資、職稱、部門、辦公地點等都先存入人事部門的人事管理軟件系統，然后信息技術部門為其在內部網絡活性目錄(Active Directory)內設立一個個人帳號。目前各種個人用戶帳號已增長到10幾萬個，只有幾個研究Linux的員工沒有帳號，因為公司要求他們忘掉Windows，完全從競爭對手的角度看問題。

5）微軟購買外部技術領先公司的產品如：CLM,MSMS,GROOVE,CRM,HOTMAIL等；而智能卡視窗操作系統被外賣。

6）“微軟離破產永遠只有18個月”——比爾.蓋茨

7）Moral Budget叫做"士氣經費"。

8）CHIME即微軟華人協會(Chinese Microsoft Employee)，成立于1992年。團體活動之影響非常廣大，為"明日中華教育基金會"進行募捐；舉辦春節聯歡晚會、中秋節晚會等。2006年4月19日是歡迎國家主席訪問微軟的日子，書中的P285頁記載了這樣的一段：”在胡主席的車隊來到前的幾個小時，有些『氵發車侖工力』成員開始在通往微軟總部的路上樹立一些'抗議'的宣傳橫幅......”

9）壞老板：員工不是離開公司，而是離開老板。（Bad Bosses: People Don't Leave Companies,They Usually Leave Bosses。)

10）全球第四大網站：microsoft.com，全球第三大網站：msn.com

11）微軟7大研發部門包括：Windows + Office + Server & tool（如SQL、Exchange、Biztalk、.net Framework、visual studio、Team system等）+ Windows Live + 移動和嵌入 + 家庭娛樂 + 商業方案。

12）微軟產品開發團隊的組織結構：

MSF微軟解決方案架構（Microsoft Solution Framework）
Feature Crew功能組
Virtual Team虛擬團隊
遠程團隊

團隊協作形式：電子郵件 + 開會 + 電話 + 上門交談 + IM + 走廊交談 + Bug Triage

Gtalk终于可以帮我传送文件

7/29/06

自GtalkBETA测试版推行后就一直在用,但不是直接从Gtalk登陆,而是让MIM自动登陆;所以看到Kenwong上的消息才知道Gtalk又升级啦;这次增加了我想要的文件传输功能!!

因为平时的工作一直离不开跟同事互传文件,用EMAIL太慢甚至可能还被弄丢,所以平时的transfer都寄托于MIM;MIM的确是不错的开源IM工具,可以定制ICQ,QQ,Y! messenger,Gmail和gtalk,应用比较广的;但是有时侯跟对方交流的时候还是不很通畅,发送的消息出现频繁阻塞;所以传文件比较快的个人认为还是网易的popo吧,但都极少人用;

这次Gtalk新推的File Transfer一定会不能错过;及时登陆后果然是出现"Google Talk Updates"的提醒信息!

官方图片File Tranfer:

不仅仅是文件传输功能增加了,而且还有另外2个应用:Voicemail(语音邮件)和music status(音乐状态);

后者就是在我们用Music Player播放节目和音乐时,自己的联系人也可以同时收听,在其他的IM工具里面也有这个娱乐功能;目前gtalk支持的music player有如下4种:
iTunes Windows Media Player
Winamp
Yahoo Music Engine

官方图片:

语音邮件,当然要电脑的硬件支持先,比如声卡和麦克风;操作的时候Kenwong还提及在Gmail里如何管理这种邮件的方法,值得学习一下!!

Firefox代言人.機器人

4/9/08

Firefox越來越火，難怪狐迷們叫她"火狐"呢！

而關於她的神話故事也在圈中傳播開來了，不過是以"代言人"的形象出現的。（要感謝"閱微堂"給我帶來了"水木社區"上的2個鬼故事，呵呵！）

另一個更加詭異事件是Firefox被想象為"機器人"，將於2008年降臨到地球，破解這一密令通道的方法是按照"天涯博弈"所說的：在Firefox 3 Beta 5的瀏覽器地址欄輸入指令（about:robots）即可。



Firefox 3 Beta 5自開放之日起已經有幾天了，雖然修補了一些漏洞，但亦有人發現使用時頻繁出錯的現象，而Firefox 3 的最後正式版預計將在6月發布。（簡體中文版下載）

今日推薦我那天（大概4月2日）發現的一個在線瀏覽器安全測試：Browser Security Test，特此用它來測試一下Beta 4的安全性（在2號這天我還在用Beta 4的呢！），當然它也適用於IE瀏覽器。（現在還有人用IE6嗎？如果有的話，那肯定是屬於這裏所指的5種類型之一。）

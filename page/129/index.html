project twitboo and twittersina

【推薦】圍著脖子推

踏破鐵鞋無覓處，得來全不費功夫啊！

今早遇上一個非常不錯的“推特”同步工具，可以直接從我的Gtalk客戶端同步信息到twitter和FriendFeed的網頁上，甚至是新浪圍脖，嘀咕網，以及人人網上的賬戶下的網頁信息，至少適合于我最好不過的啦！非常感謝原創者@StevenWnag的辛勤勞動！！

由于twitter->FriendFeed和FriendFeed->Twitter是同等的，所以，以后如果FF的Gtalk機器人斷線了，還可以用“圍著脖子推”的機器人：twitboo@appspot.com，真是互通有無了。

之所以要強烈推薦：圍著脖子推，是因為她是基于GAE托管的，而且也是在Google Code上開源了的，只不過她是由Java開發的，可以進入下面的地址進行了解：

http://twittersina.googlecode.com/

除了使用她的Gtalk機器人外，還推出了一個web版的同步工具，參見如下地址： http://twittersina.appspot.com

進入頁面上端的“管理同步源”，然后填寫你的twitter推特.sina圍脖.人人網.嘀咕網和做啥網的帳號及密碼后，就可以發布信息了。雖然“同步源”的支持不夠多，但我覺得僅支持twitter就已經足夠了。

如果用twitboo@appspot.com機器人來發布，可以多2个同步源：F5和開心網。而且客戶端上還可以顯示twitter上的更新，以及用命令行來跟推友互動（reply），這樣在沒有用代理的情況下，也可以查看web/twitter上的信息，是不是更方便呢？！

附：围着脖子推机器人V1.0帮助

绑定账号：-bind -同步源代码 账号 密码 如：-bind -t myn0 myp0 -s myn1 myp1 同步源代码： t-Twitter, s-新浪围脖, r-人人网, d-嘀咕, z-做啥,k-开心网, F5-Follow5 删除绑定：-remove 查看绑定状态：-status 选择同步源发布信息：-t -s -r -d -z -k -f5 mymsg [-t -s -r -d -z -k -f5] 可选择一个或多个进行组合 开启Twitter更新提示：-ton 关闭Twitter更新提示：-toff 查看Twitter最近20条更新：-thome 查看Twitter最近20条回复：-treply Twitter Reply：-treply statusID @name mymsg 如：-treply 1234567890 @Jim This is Twitter. 查看帮助：-help 注：以非"-"开头对话，即向所有同步源发布消息

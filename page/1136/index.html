Messenger's Birthday

7/20/06

每天都离不开messenger,但是可曾想过她的出生日期呢?我也是偶然逛到她的官方blog才知道的! 源自:Inside Windows Live Messenger

1999年7月22日是 messenger的诞生日!而这个月的22号正好是7周年纪念日啦!现在已经变身live 版本了,所以在这个big day即将到来之季,别忘了说声:""HappyBirthday, Windows Live Messenger!"

在7周年纪念活动中推出了免费下载动漫表情和背景图片的服务;

1)Emoticons,3H公布,下载仅支持IE6+浏览器和activeX控件允许;

2)表情和背景下载包,由Kiwee提供;

3)由Quebles提供的

除此之外,还有一段祝贺的声像记录在youtube上面,让我们唱响"Happy Birthday"吧!

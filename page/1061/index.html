一千万个问题

12/1/06

Google Answers一个收费模式的问答服务,已经宣布不再接受用户的新问题; 下面就是正页上面的告示:

Google Answers is no longer accepting questions.

We're sorry, but Google Answers has been retired, and is no longer accepting new questions.Search or browse the existing Google Answers index by using the search box above or the category links below.

而与GA收费相反的免费服务有Live Q&A,雅虎"知识堂",新浪"爱问"等; 哦,肯定不会忘记还有个"百度知道",而且它正在掀起一个推广活动! "寻找第一千万个问题,参加幸运踩蛋"

"百度贴吧"三周年啦! "音乐掌门人"一周岁啦! 官方空间提示将要发布硬盘搜索2.5版本! 另外,百度还要推出"博客搜索",而且还是"三搜"即将上线!

好似很热闹! 那么独霸国内互联网江湖的日子岂不是越来越近了呢?

我只想了一个问题而已,第一万个也许临不到我!!

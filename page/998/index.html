你服务.我体验

3/30/07

IM and Email以及收藏夹大概成了大部分网友们每天必用的工具,如同一日三餐;然而围绕这些网络应用,各个服务提供商可谓耗用了不少心机,也让我们的体验百花齐放;我的口号是"你服务,我体验"!再好的免费服务,只要适合自己就基本上不舍不离!

1)你要用Twitter吗？(by 方军) - 思维的乐趣BLOG
2)如何通过MSN Messenger/GTalk玩维众 - Andy's blog
3)希望IM与Ing结合
4)Free Skype Rss Bot Widget(Anothr)

上面四个方面的话题都与IM联系工具(如:WLM.Gtalk.Skype)有关;现在的玩法都倾向于将自己所想说的内容从IM发送到WEB上面,以使自己熟悉的朋友知道自己在做什么,想什么;反过来,也能将Blog或Web上面的Rss内容转移到IM中;信息(eg:Who+When+Where+What)就这样互动而活跃起来!尽管Twitter/Ing/Saying都有字数限制,但已经足够了;

Anothr可以订阅Rss,让用户与被订阅的信息之间的距离更近一步;现在已经支持Gtalk添加Anothr机器人,并且能够生成Button或Widget放于自己的Blog上面;亦可以作为浏览器中的Bookmaklet来使用;

Ing和Saying(呻吟)在社区的概念上先下一城;Saying可以与Gtalk结合了;将Gtalk上面的说话内容发送到社区上,如果愿意共享,那么其他人通过简单的命令操作就能看到自己的所说所想;

Twitter开放了API后,可以看到很多Meshup例子;比如在twittervision上面可以从Google Map上观察到各地朋友在做什么;看着一段段话语象肥皂炮纷纷冒出来的样子,就感觉网络就是动的;

1)Pengbone’s Weblog|如你所知 ?Google Notebook变脸
2)百度搜藏首页今日全新改版 增加帮助与技巧 - 百度粉丝网

对于Google Notebook这款在线收藏夹虽然常驻在Firefox中,但是我却很少去用了,顶多是存放书签的地方;不过看到它现在改变了UI也仍是在关注之中;百度的收藏夹我还是先不用了;基本上有了365Key和Del.icio.us就够用的啦;这次想不到二者的变脸几乎发生在同一时段;

1)sohu通行证
2)好消息！网易免费邮全面升级附件至20兆!
3)GSeeker - 庆祝十周岁生日　Yahoo!5月起提供无限邮箱容量
4)cnBeta.COM - Windows Live Hotmail M10 正式上线

"通行证"在Msn服务中少不了的,在新浪和网易上早已有之;而这个时候sohu推出了通行证,究竟有什么新颖之处呢?能吸引我的地方就是在输入自己的id的时候,登陆窗口会连带显示id后缀,后缀包括自己曾经注册使用过的邮件.同学录.搜狗等帐号;在Yahoo!宣布邮箱大小无限量的时候,网易这一中文邮箱品牌也宣布放宽对附件上传大小的限制;以前还一直叫唤Live Mail,但是现在我们又得改口称之为"Live Hotmail"了,真不知道有多大的意义,反正就是慢就一个字;

直到今日,我用的邮箱依然是163和Gmail;马上要到愚人节了,不知道这天Google会唱什么戏给大家看!

1)雅虎中国推出群组产品-WhaDonce|王华东-搜狐博客

Yahoo!早就有了个Groups,现在雅虎中国也推出了群组叫Club,名字也好象没有叫好!我看比起Google群组还是欠缺名气;这个Club能跟百度贴吧站在同一高度上吗?我不信在"百日维新"的日子里那些技术人员就搞这些东西出来!

2)百变版百度上线 创建中国人百变的搜索生活！度百空间 外表变来变去没有什么意义的,还是提高搜索内容的质量吧!

3)Microsoft Live Labs: Deepfish - What is Deepfish?

相传iphone和Google Phone的消息已不少,这主要是硬件上的光彩;而软件巨头微软把目光放在手机(移动设备)浏览软件上;28日微软在实验室上发布了Deepfish的预览版,由于是Beta测试期,还有下载数量限制,在Windows Mobile 5.0+ 的设备上可以安装;目前还未能支持ActiveX控件, AJAX, cookies, Javascript, and HTTP POST等;不过,它的突破在于为手机等移动设备持有者浏览页面提供了与PC桌面上使用浏览器相似效果;

"数字信息","移动革命"的词语一直在伴随着我们的生活;这个网络时代,只有我想不到的!

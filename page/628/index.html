谷歌瀏覽器中文測試版

9/3/08

噢！終於見到了Google Chrome的面孔：十分簡潔、速度很快的瀏覽器。

【李開複表示，瀏覽器是一個戰略性的方向，但不一定是戰略性的產品。他指出，在雲計算的未來，瀏覽器將是非常核心的平台，未來用戶上網、接觸雲計算，無論是在PC、手機或其它硬件上，都需要強而有力的瀏覽器。】（參考閱讀：Google開發瀏覽器的原因)

李開複做客新浪獨家訪談的時候稱：穀歌瀏覽器將同步進入中國。沒有想到真的這么快就有了中文測試版（適用於Windows XP SP2或Vista系統）！

今早就迫不及待地下載和安裝，現在經過簡單試用，感覺還不好說啊！如今，除了Firefox 3和K-Meleon CCF ME外，我又多了一種選擇了。

得到的安裝文件不是很大:474KB,版本號：1.2.131.11

安裝過程中導入了我的Firefox書簽和設置。默認安裝在如下目錄：
C:\Documents and Settings...\Local Settings\Application Data\Google\Chrome

但是，當我打開了3個標簽頁後，windows任務管理器中出現3個進程，這個我是不太喜歡啊，怎么像IE了呢？！

而對網站管理員或開發人員，Chrome的快捷功能做得很到位：在地址欄右側含有【控制當前頁】和【自定義設置】項目（網頁菜單），諸多的功能和設置都可以獲得。可以利用【網絡檢查器】、【任務管理器】、【JavaScript調試程序】進行網站測試。

Chrome內置了Gears，能讓網站管理員充分利用離線存儲等 API 功能；使用了全新的 JavaScript 引擎 (V8)，新引擎的運行速度遠高於現在的 JavaScript 解釋器；Chrome是建立在 WebKit 的基礎上，當推出 CSS3功能時，Chrome用戶即可受益於添加到WebKit的這些CSS3 功能。

Chrome和Firefox/Safri等瀏覽器不支持ActiveX控件。

Chrome遇到"崩潰"的情況時，可使用"恢複"網頁的方法啟動上一次的網頁。

目前,Chrome支持正確顯示網頁所需的最熱門插件，包括 Flash、Acrobat Reader、Java、Windows Media Player、Real Player、Quicktime 和 Silverlight。

Chrome附加的搜索引擎包括：百度、中國雅虎、搜狗、搜搜等等。當然默認的是穀歌(google.cn)搜索，我還沒有找到把它修改成預設Google.com的方法。

總之，Chrome讓我覺得"麻雀"雖小，但是"五臟"俱全！

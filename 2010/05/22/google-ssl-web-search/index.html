5月 22 2010

google ssl web search

二三事

周末啦（今天是5.22），記住今晚（5.23凌晨02：45）歐冠的決賽不能錯過了。

早上上機用Google.com的時候，注意到首頁logo變成了動態游戲畫面，原來是為了紀念一款叫“吃豆人”的游戲誕生30周年。（參照：貓貓對Pac Man簡單分析）

Google的工程師們在普及藝術.科技文化方面帶了個好頭啊，往往在出其不意的情況下推出我們平日里幾乎不曾接觸的新知，這亦是一種留住用戶的招數，UE（用戶體驗）的提升也增強了更多的貼性。

另外，還有好消息是：google加密搜索beta發布了！

https://www.google.com

雖然目前只支持web搜索，但加了(https://)的好處是不用多說了，至少現在你可以輸入某些“敏感詞”（eg:“胡/羅/卜”.“天/安/門"等）試一試結果是否暢通。

如果你用chrome瀏覽器，可以參照“小眾軟件”上介紹的辦法實現自動使用https搜索。

但，想起那邪惡的GFW，心里不免擔心地問：Google SSL Search能堅持多久呢？

2010年的Google I/O大會發布了不少新技術和產品以及開放了不少API，顯然成為了這2天的熱文，VP8+WebM+Adroid+Froyo+Chrome Store+GTV+Web Font等等雄霸IT頭條，真令人應接不暇。

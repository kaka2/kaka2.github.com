desktop

雖然硬件也是計算機（微機）重要的一部分，但用電腦其實就是用軟件。不過，雙方的作用是相互的，並行計算，多核技術等都與軟件和硬件的發展分不開。

一。字符界面(CLI)時代

最早接觸的機器還是286/386的，還沒有鼠標，存儲只能用1.44MB的軟盤。後來有了486的機器就是神了，有個32M的內存那更是高檔機啦！現在的機器配置則進步得多了，動不動就是2G內存，雙核，SSD，還有刻錄機，液晶屏等；當初只能通過主板/內存跳線，DIY進行“烤機”來增加性能。

96年左右，在學校機房裡用鍵盤敲出自己的名字那算是高人，那時有PCDOS/UCDOS和WPS，能夠在畢業前打印出簡歷那就了不起啊！後來在工作時期見識了Netware網絡系統，當時工作機都是無盤的，只能使用DOS命令，運行Lotus1-2-3和Lotus Notes，還有QBasic編程以及QEdit編輯器；過了一段時間就是微軟的Windows3.1和Windows95/98視窗操作系統遍地開花。早期接觸的數據庫是DBase和FoxPro。當時見過一些工程師們在微機上寫一些代碼，然後就可以測試一些"火牛"（磁力線圈）的參數，用來判斷成品是Pass還是NG，真的很崇拜他們啊！

二。圖形界面(GUI)時代

軟件有千萬種，不可能一網打盡，只能說選擇適合自己的最重要，畢竟她們只是一種工具。當然除了辦公主要用的Word/Excel等外，不過更我喜歡一些輕量級的東東，下面的是留存在系統內仍未被卸載的部分軟件：

C:\Program Files

1.郵件：Mozilla Thunderbird（雷鳥）

2.MrZhang：張生翻牆crossthewall，你懂的喔。

3.集中式版本控制：TortoiseSVN

4.拼音輸入法：Google PinYin2

5.IM即時通訊工具：Google Talk

D:\Program Files

1.解壓縮：7ZIP

2.防火牆殺毒：AST超級巡警

3.截屏：Faststone Capture

4.PDF：Foxit Reader

5.冰刃：IceSword，進程/註冊表管理利器，木馬殺手。

6.離線blog發布管理：kicoy

7.FTP上傳：LeapFTP

8.下載利器：Orbit

9.PDF轉換：Pdf2Eex，共享軟件

10.動態域名解析：PeanutHull5，花生殼域名工具。

11.網絡編輯超級工具箱：Editor，快速格式化，繁簡互轉等。

12.Unix/telnet Client：fterm，上BBS，操作Unix的終端工具。

13.p2p超強工具：Foxy，流行於港台地區。

14.視頻：SopCast，播放網絡電視的幫手。

15.火狐瀏覽器：Mozilla FireFox 3

16.多種IM聚合：pidgin

17.K-Meleon中文: K-Meleon CCF，速度夠快。

18.TCP半開連接數破解補丁：TCP-Z

19.免費VPN：UltraVPN

20.郵件：網易閃電郵

21.SSH/代理：Puff，可通過Buzz最終最新版本信息。

23.系統優化：Zwuqi，原名Z武器，現名：魯大師。

26.SSH：PuTTY

27.分佈式版本控制：TortoiseHg，類似的：Mecurial。

28.源碼管理：Git，亦是分佈式版本控制工具。

29.源碼編輯：Notepad++

D:\

2.LAMP套件：XAMPP，服務於本地的WordPress和Discuz!體驗 。

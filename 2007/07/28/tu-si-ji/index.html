兔斯基

7/28/07

“兔斯基(TUSKI)”,我居然用谷歌拼音能夠把這個單詞打出來，它還是一個流傳網絡間的詞匯呢，其風頭甚至賽過昔日的“洋蔥頭”；偶然從改版后的新浪論壇瀏覽到“銳話題”，不成想卻發現了一份有趣的試卷---2007“兔商”等級考試。除了“智商”.”情商“."健商"等之外，也許你和我一樣頭一次聽說還有一個“兔商”的詞吧？

下面是試題及答案，看完你就明白屬于“兔斯基”的一些有趣故事了！

一、选择题（共5题，每题4分。）

1 兔斯基为什么叫这个名字？(答案C)
A 兔斯基是俄国籍兔子。
B 兔斯基的作者受大片中“崔永元夫斯基”的影响。
C 作者受短篇小说《请用“S”拼我的名字》的影响，迷恋“斯基”。
D 因为作者的名字叫兔斯基。

2 兔斯基究竟是什么？(答案A)
A 喜欢胡思乱想，满腹牢骚的兔子。
B 流氓兔
C 自杀兔
D 团圆兔

3 图中的这个表情是什么意思？(答案D)
A 抽筋
B 悲痛难当
C 抒情 D 得意忘形

4 选项中哪个不符合兔斯基性格？(答案C)
A 颓废
B 闷骚
C 笨蛋
D 放纵

5 现在兔斯基系列有多少个表情？(答案B) A 20多
B 40多
C 50多
D 100多

二、填空（共5空，每空5分。）

1 作者王卯卯的“卯”字拼写成＿Mao＿，王卯卯的性别是＿女＿。
2 兔斯基的性别是＿男＿，婚姻情况为＿未婚＿，身高约＿1.2M＿。

三、判断题（共2题，每题10分。）
1 兔斯基边开口说话，边用柔软若面条的手臂和身体的扭动表露自己的心情。 (错,兔斯基从不开口说话。)

2 兔斯基喜欢喝牛奶，吃奶酪，不喜欢吃胡萝卜。
(正确。)

四、名词解释 （共3题，每题5分。）

1、兔斯基主义：
像兔斯基这种思维方式，这种生活态度；

2、兔斯基：
一个小兔子用柔软若面条的手臂和身体的扭动表露自己的心情。有点懒散，有点颓废，时常茫然，容易失落，喜欢胡思乱想，经常满腹牢骚。

五、问答题（共2题，每题10分。）

1、为什么兔子平时没有嘴巴呢？
张不开嘴巴，眯着眼睛，兔斯基的特征，源于为他塑造的性格，以这种有点懒散、有点颓废的性格基调下，他的言行举止似乎理所应当就是这个样子了。 受到强烈刺激、震惊的时候，兔斯基偶尔会把眼睛睁开，嘴巴张开，作嗔目结舌状。

2、你愿意用哪些词来评价这只兔子呢？
仁者见仁智者见智，此题无标准答案……

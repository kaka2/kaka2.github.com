Firefox升級了啊

11/28/07

剛剛才看到的提示，之前也有消息透露會在近期有更新，沒有想到正趕上了！新版本：2.0.0.10，下載各種安裝版點擊這里！

提示：



修復的高危(high級別）, 漏洞有：

MFSA 2007-39 Referer-spoofing via window.location race condition
MFSA 2007-38 Memory corruption vulnerabilities (rv:1.8.1.10)
MFSA 2007-37 jar: URI scheme XSS hazard

想不到中國Yahoo!搜索小組公布了一個很酷的Firefox插件——SearchAll（我搜去），它是一款搜索引擎并行比較工具，只要在工具欄上輸入想查的詞，就會查詢到自定義組合的多家搜索引擎（如谷歌、雅虎、百度、淘寶、Amazon等）所獲得的結果。對于“火狐”迷來說，還是愿意去嘗試一下為好！

再補充其他3個見聞吧：

1）早上刷新本blog頁面的時候，看到上面Google Adsense的廣告框中多了一行文字鏈接：『顯示更多廣告』；冥冥記憶中上次是改變了整個廣告框的可點擊區域，怎么又來神秘的一次？只可惜今次我沒有截圖啊！

2）在blogger的編輯后臺，看到也有條12.26日更新信息；即是在blog中可以從youtube里面共享視頻。

以前是可以上傳視頻至blog，現在已經可以跟Youtube集成啦！是的，選擇blogger.com作為發布平臺是個極好的idea,因為她一直在改善和改變，更貼近大家的使用體驗！難怪，在2007年10月份美國博客網站排行榜中占第一位的就是她！

3）當然，還有一個令人沉痛的社會現實話題也要關注的，就在【周老虎】迸發【第二季】熱潮的時候，有一個叫”肖志軍“的不幸男子進入網際討論！總之，越是這類新聞看多了，越覺得活得很累啊！錢錢錢錢，沒有錢是萬萬不能的嗎？

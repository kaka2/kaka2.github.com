关键词：收购和Gears.Surface等

5/31/07

要到6月了，意味著07年快过一半啦！Google和微软两大巨头也忙着推出他们的新服务或产品，或者忙于收购其他国内公司来扩充自己的业务！

这样的事情好是好，但也难为了那些追踪新鲜爆料的Geek或blogger们啊！我虽称不上甚么Geek之类，但自己订阅的rss也不少，看了很多新科技新信息就只有劳驾收藏夹来帮忙哦！

Google和微软在这几天发布的产品和网络新闻真不少！

先说Google的吧，也许有的是我不太感兴趣而引致些许遗漏的，但不要紧，毕竟这里记载的是我自己的所见所闻而已！

1)Google收购西班牙照片共享网站Panoramio

这项服务在使用Google地球、地图产品的时候就可以发现的；如果不熟悉的话，还可以看看之前Willamlong所发的"照片共享服务panoramio和flickr的对比评测"一文!

2)Webleon称Google Gears为"互联网的离线发动机",首先推出的应用就是离线的Google Reader;可以想见今后说不定还有离线Gmail,如同微软的email桌面工具Windows Live Mail Beta;

详情见正式页面:Google Gears (BETA)

通过以下3种JavaScript API提供离线应用: Cache and serve application resources (HTML, JavaScript, images, etc.) locally (本地存储及处理应用程序资源)

Store data locally in a fully-searchable relational database (从全搜索的关系数型据库中实现本地存储数据)

Run asynchronous Javascript to improve application responsiveness (运行异步脚本来改进应用程序的互动)

还可以参照这里,还有这里;

3)Google Mashup Editor 基于ajax的开发框架，包括三部分:editor,Feed Browser,Sandbox；类似于Microsoft PopFly 和Yahoo Pipes的Mashup工具;

更多信息可参照Gseeker的介绍!

接下来就是微软的消息！

1）Microsoft Surface http://www.microsoft.com/surface/

真正的“桌面”电脑 微软推出触摸桌面电脑,开发代号为milan；触摸屏的应用相信在车站.银行都有见过；但微软把这种技术应用在电脑上确实比较超前，从视频中看它更像是大家所想到的一张“咖啡桌”，也被称作“平面计算机”！

更多介绍见这里 !

2)微软欲推站点统计服务(代号“Gatineau”)，最初的消息来自A Little Blog !

3)Windows Live Betas 网站 新增3个成员;

4)微软60亿美元收购网络广告公司

早上还在逛last.fm上面的歌曲，后来看IT新闻，发现CBS已经将它收购了，2.8个亿啊！last.fm真不愧为是革命性的音乐社区；

其他的收购案例

1)eBay以7500万美元收购浏览推荐网站,由先前的传闻，到收购被确认，来得太快啦！ StumbleUpon是一个网络评级网站，提供Flickr，MySpace，YouTube上热门评论的页面内容，用户可以进行投票。目前，这家网站拥有230万用户。平时如果觉得所见到的网站太过平凡，那么在这个社区上可发现很多有趣的网站；最好是在Firefox中安装它的插件来配合使用！

2)新闻集团3亿美元收购多媒体分享网站Photobucket

这一收购行动让新闻集团福克斯互动媒体（Fox Interactive Media）下属的MySpace及其他站点，可使用Photobucket的照片及视频分享技术，同时Photobucket也可获得福克斯的资源加速工具软件的开发。
